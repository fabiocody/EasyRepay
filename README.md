# EasyRepay

## Installation

```bash
$ npm install
$ cd angular
$ npm install
```

## Running the app (development)

```bash
$ npm run start

# watch mode
$ npm run start:dev
```

## Running the app (production)

```bash
$ npm run build
$ cd angular
$ npm run build
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
